# Spec: Add Reusable Intelligence Section

## User Stories

### US1: Learn to Create and Use Reusable Intelligence

- **As a** participant
- **I want** to access a guide on how to create and use reusable intelligence, specifically with Claude Code Subagents and Agent Skills
- **So that** I can effectively apply these advanced techniques in my project and enhance my capabilities.

#### Acceptance Criteria

- A new documentation section is created for "Reusable Intelligence."
- The section contains a new page explaining the concepts of Claude Code Subagents and Agent Skills.
- The page provides a step-by-step guide or examples on how to create and use them.
- The new section and page are accessible from the main navigation sidebar.
- The content is clear, well-structured, and provides practical knowledge.
