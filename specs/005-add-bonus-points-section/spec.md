# Spec: Add Bonus Points Section

## User Stories

### US1: Understand Bonus Points Opportunity

- **As a** participant
- **I want** to find clear information on how to earn up to 50 extra bonus points
- **So that** I can understand the requirements for creating and using reusable intelligence with Claude Code Subagents and Agent Skills and aim for a higher score in the book project.

#### Acceptance Criteria

- A new documentation page is created explaining the bonus points system.
- The page details the criteria for earning the 50 bonus points.
- The page provides guidance or examples on creating "reusable intelligence," "Claude Code Subagents," and "Agent Skills."
- The new page is accessible from the main navigation or a relevant section like "Assessments."
- The content is well-structured, clear, and easy to understand.
