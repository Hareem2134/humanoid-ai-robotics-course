# Spec: Remove Bonus Points Section

## User Stories

### US1: Remove Bonus Points Opportunity

- **As a** course administrator
- **I want** to remove the bonus points page and any references to it from the course book
- **So that** the information about bonus points is no longer available to participants.

#### Acceptance Criteria

- The `bonus-points.md` documentation page is deleted.
- The link to the "Bonus Points" page is removed from the sidebar navigation.
- The Docusaurus site builds successfully without any errors related to the removed page or broken links.
- The bonus points page is no longer accessible on the live or locally served site.
