---
title: Glossary
---

This section contains definitions of key terms used throughout this book.

**ROS 2:** The Robot Operating System, version 2.

**URDF:** The Unified Robot Description Format.

**Gazebo:** A popular robotics simulator.

**Unity:** A game engine often used for photorealistic simulation.

**NVIDIA Isaac Sim:** A robotics simulation platform from NVIDIA.

**VSLAM:** Visual Simultaneous Localization and Mapping.

**Nav2:** The navigation stack for ROS 2.

**VLA:** Vision-Language-Action model.
