---
title: Learning Outcomes
---

Upon completion of this book, you will be able to:

*   Explain the architecture of ROS 2 and build distributed robotic systems.
*   Create and simulate robotic systems using Gazebo and Unity.
*   Implement perception and navigation algorithms using NVIDIA Isaac Sim.
*   Integrate Vision-Language-Action models with humanoid robots.
*   Design and program humanoid robots for a variety of tasks.
