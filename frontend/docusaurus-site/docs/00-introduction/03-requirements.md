---
title: Hardware and Software Requirements
---

To follow along with the examples in this book, you will need the following hardware and software:

**Hardware:**

*   A computer with a dedicated NVIDIA GPU (for NVIDIA Isaac Sim)
*   (Optional) A physical humanoid robot

**Software:**

*   Ubuntu 22.04
*   ROS 2 Humble
*   Gazebo
*   Unity
*   NVIDIA Isaac Sim
